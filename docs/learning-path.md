# Learning Path

This repository is designed to be read in layers:

1. `templates/` and `utils/` to understand shared conventions.
2. `01-basics/` through `03-networking/` for strong Bash fundamentals.
3. `04-ci-cd/` through `06-kubernetes/` for application delivery automation.
4. `07-cloud/` through `09-security/` for infrastructure and compliance workflows.
5. `10-sre-usecases/` and `projects/` for incident-driven and multi-step operational systems.

Teams can also assign folders by role:

- Platform engineers: `04-ci-cd/`, `05-docker/`, `06-kubernetes/`
- Cloud engineers: `07-cloud/`
- Security engineers: `09-security/`
- SREs: `08-monitoring/`, `10-sre-usecases/`, `projects/monitoring-alert-system/`
